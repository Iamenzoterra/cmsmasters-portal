## What Was Implemented
Created `apps/command-center/ui/Modal.tsx` as a controlled React dialog component. The component renders a fixed full-screen backdrop (`bg-black/50`) with a centered `zinc-900` panel containing a header with title and close button, and a body for children. Escape key dismissal, backdrop click close, and body scroll lock are all handled via a `useEffect` tied to the `open` prop with proper cleanup on unmount/close. Accessibility is provided via `role=dialog`, `aria-modal=true`, and `aria-labelledby` linked to the title `h2`. The modal is rendered via `createPortal` into `document.body`.
