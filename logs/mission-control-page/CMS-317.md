## Issues & Workarounds
`components.json` at runtime has shape `{ lastScanned: string; components: ComponentSummary[] }` rather than a plain array. `getComponents()` types the return as `ComponentSummary[] | null` but actually returns the wrapper object. Added runtime guard in `page.tsx`: `Array.isArray(componentsRaw) ? componentsRaw : (componentsRaw as ...)?.components ?? null`.
