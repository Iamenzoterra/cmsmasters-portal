## What Was Implemented
Created `ActivityFeed.tsx` as a `'use client'` presentational component that accepts a `tasks: Task[]` prop. It derives a sorted feed of up to 10 events from tasks with `completedAt` set or `status !== 'todo'`, normalizes each to a `TaskEvent`, renders a vertical list with status icons (CheckCircle/Loader2/XCircle), relative timestamps via `getRelativeTime`, and navigates to `/phases?task={id}` on click.

## Key Decisions
| Decision | Chosen | Why |
|----------|--------|-----|
| Status icon mapping | CheckCircle (success), Loader2 animate-spin (active), XCircle (danger/muted) | Matches plan spec and design token colors |
| Timestamp fallback | `completedAt ?? updatedAt ?? ''` | Covers both completion and status-change events |
| Empty state | Centered `<p>` inside component | Simple, no extra wrapper needed |
| No default export | Named export only | Follows project convention |
