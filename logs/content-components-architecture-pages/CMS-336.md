## What Was Implemented
Created `ThemeStatusTable.tsx` as a client component accepting a `themes: ThemeEntry[]` prop. The component renders a searchable, filterable table with a status dot, name, docs/plugins/features counts, hero image indicator, and last updated timestamp per theme. Clicking a row opens a slide-out panel (same structural pattern as `TaskDetailSheet.tsx`) showing drill-down details and a Supabase link placeholder. Status logic: green (published + docsCount≥5 + hasHeroImage), yellow (draft with some data), red (empty/zero docs).
