## What Was Implemented
Created `KeyboardShortcuts.tsx` as a `'use client'` component that registers a global `keydown` listener via `useEffect`. Keys 1–6 push routes via `useRouter`. Pressing R/r shows a 2500ms toast notification. Escape dispatches a `cc:escape` custom event. All `window` references were replaced with `globalThis` to satisfy the `unicorn/prefer-global-this` ESLint rule. Input-field guard skips shortcuts when focus is in INPUT, TEXTAREA, SELECT, or contentEditable elements.

## Key Decisions
| Decision | Chosen | Why |
|----------|--------|-----|
| `window` vs `globalThis` | `globalThis` | Project ESLint enforces `unicorn/prefer-global-this` (error level) |
| Toast timer | `useRef` for timer handle | Allows cleanup on unmount and reset on repeated R presses without stale closures |
| Object injection warning | `as string` cast on ROUTE_MAP access | Silences sonar warning while keeping type safety |
| No default export | Named export `KeyboardShortcuts` | Per project conventions (no default exports except Next.js pages) |
