## Invariants

- **tokens.css is auto-generated by `/sync-tokens` skill.** Do NOT edit manually. Source: Figma files via Plugin API.
- **HSL values stored WITHOUT wrapper.** Token value is `228 54% 20%`, not `hsl(228, 54%, 20%)`. This is the shadcn convention.
- **No build step.** Consumers import TypeScript directly from packages/ui/src/. No compiled dist/.
- **Three-Layer structure:** `primitives/` (atoms) → `domain/` (business composites) → `layouts/` (page shells). Currently only primitives has real components.
- **portal-blocks.css provides `.cms-btn` system** used by ALL portal blocks for interactive elements.
- **animate-utils.js exports 5 behavioral animation utilities** (1.5KB) — scroll-driven entrance + behavioral JS per ADR-023.
- **tokens.responsive.css is machine-generated by `tools/responsive-tokens-editor`** (WP-030, ADR-025 Layer 1). SOT: `packages/ui/src/theme/responsive-config.json` (sibling file). `/sync-tokens` does NOT touch either file. Direct edits to `tokens.responsive.css` are overwritten on next Save — open the editor (`:7703`) instead. Currently 22 fluid tokens (10 typography + 11 spacing + 1 special) + 3 discrete container `@media` blocks; conservative V1 defaults preserve desktop static (Phase 0 ruling 1). (WP-024 scaffold → WP-030 populated / ADR-025)

## Traps & Gotchas

- **Tailwind v4 font-size hint required:** `text-[length:var(--text-sm-font-size)]` NOT `text-[var(--text-sm-font-size)]` — without `length:` hint, Tailwind interprets the value as a color.
- **Tailwind v4 bare var for sizing:** `h-[--button-height-sm]` NOT `h-[var(--button-height-sm)]` — `var()` wrapper causes Tailwind to skip class generation.
- **Color needs hsl() wrapper in classes:** `bg-[hsl(var(--primary))]` — because the token stores raw HSL triplet without hsl().
- **Command Center does NOT use these tokens.** CC has its own dark theme in `apps/command-center/tailwind.config.ts`. Changing tokens.css does NOT affect CC.
- **button.stories.tsx exists** but Storybook is not configured project-wide — stories are for reference/documentation only.

## Blast Radius

- **Changing tokens.css** — affects Portal, Studio, Dashboard, Admin (everything except CC)
- **Changing portal-blocks.css** — affects all portal blocks' interactive elements (.cms-btn)
- **Changing animate-utils.js** — affects blocks using behavioral animations
- **Changing button.tsx** — affects every app that imports Button from @cmsmasters/ui

## Recipes

```tsx
// Import Button:
import { Button } from '@cmsmasters/ui'
<Button variant="primary" size="sm">Click</Button>

// Use tokens in Tailwind:
<div className="bg-[hsl(var(--primary))] text-[length:var(--text-sm-font-size)] h-[--button-height-sm]">

// cn() utility for conditional classes:
import { cn } from '@cmsmasters/ui'
<div className={cn('base-class', isActive && 'active-class')}>
```
