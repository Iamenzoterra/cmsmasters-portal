# Figma ‚Üí CSS Token Sync

Reads all variable collections from two Figma files via Plugin API (`use_figma`), resolves aliases, converts types, applies naming rules, and writes `packages/ui/src/theme/tokens.css`.

## Source files

| File | Key | Collections to read |
|------|-----|---------------------|
| CMS DS Portal ‚Äî Obra | `PodaGqhhlgh6TLkcyAC5Oi` | semantic colors (light + dark), typography, spacing, border radii |
| Portal DS ‚Äî CMSMasters | `CLtdO56o9fJCx19EnhH1nI` | Primitives (Color/*), Semantic (Bg/*, Text/*, Border/*, Status/*, Button/*, Tag/*, Card/*, Section/*, Shadow/*, Pair/*) |

## Step 1: Read variables from Obra

Use `use_figma` with fileKey `PodaGqhhlgh6TLkcyAC5Oi`:

```javascript
// Read ALL variable collections
const collections = figma.variables.getLocalVariableCollections();
const results = [];

for (const collection of collections) {
  const modes = collection.modes;
  for (const varId of collection.variableIds) {
    const v = figma.variables.getVariableById(varId);
    if (!v || v.name.startsWith('hacks')) continue;

    const entry = { collection: collection.name, name: v.name, type: v.resolvedType, modes: {} };

    for (const mode of modes) {
      const raw = v.valuesByMode[mode.modeId];

      // Resolve aliases recursively
      let value = raw;
      let depth = 0;
      while (value && typeof value === 'object' && 'type' in value && value.type === 'VARIABLE_ALIAS' && depth < 10) {
        const aliased = figma.variables.getVariableById(value.id);
        if (!aliased) break;
        value = aliased.valuesByMode[mode.modeId] ?? aliased.valuesByMode[Object.keys(aliased.valuesByMode)[0]];
        depth++;
      }

      entry.modes[mode.name] = value;
    }
    results.push(entry);
  }
}

return JSON.stringify(results);
```

**IMPORTANT**: STRING type variables (font-family) may return empty. If so, do a second pass using `resolveForConsumer`:
```javascript
const v = figma.variables.getVariableById(varId);
const resolved = v.resolveForConsumer(node); // need a text node reference
```
Or hardcode known font families from the typography collection names.

## Step 3: Convert values

### COLOR ‚Üí HSL
```
r,g,b (0-1 floats) ‚Üí hsl(H S% L%) ‚Üí store as "H S% L%" (no wrapper, no commas)
```
Round H to integer, S and L to integer percent.

### FLOAT ‚Üí px
For spacing, radii, sizing: append `px` suffix.
Exception: font-weight stays numeric (400, 500, 600).

### STRING ‚Üí as-is
Font families: add system fallback stack.

## Step 4: Apply naming rules

| Figma pattern | CSS output | Rule |
|---------------|------------|------|
| `general/primary` | `--primary` | Strip `general/` prefix |
| `general/primary foreground` | `--primary-foreground` | Strip prefix, spaces‚Üíhyphens |
| `unofficial/ghost hover` | `--ghost-hover` | Strip `unofficial/` prefix |
| `card/card` | `--card` | Collapse duplicate: `card/card X` ‚Üí `--card-X` |
| `sidebar/sidebar` | `--sidebar` | Same collapse rule |
| `focus/ring` | `--ring` | Strip `focus/` prefix |
| Portal DS `Color/The-Sky` | `--brand-the-sky` | Add `brand-` prefix, lowercase |
| Portal DS `Bg/page` | `--bg-page` | Keep category prefix as-is |
| Portal DS `Text/primary` | `--text-primary` | Keep as-is |
| Portal DS `Border/default` | `--border-default` | Keep as-is |
| Portal DS `Status/success` | `--status-success` | Keep as-is |
| Portal DS `Button/primary-bg` | `--button-primary-bg` | Keep as-is |
| Portal DS `Tag/active-bg` | `--tag-active-bg` | Keep as-is |
| Spacing `3xs` | `--spacing-3xs` | Add `spacing-` prefix |
| Radii `rounded-md` | `--rounded-md` | Keep as-is |
| Typography `heading 1/font-size` | `--h1-font-size` | `heading N` ‚Üí `hN` |
| Typography `paragraph/large/font-size` | `--para-large-font-size` | `paragraph` ‚Üí `para` |
| Typography `Button/height-sm` | `--button-height-sm` | Keep as-is (component tokens) |
| Anything starting with `hacks` | **SKIP** | Never output |
| Anything starting with `obra-shadcn-docs` | **SKIP** | Internal docs |
| Collection `raw colors` | **SKIP** | Semantics already alias them |

**All names**: lowercase, spaces to hyphens, forward slashes to hyphens.

## Critical rules

1. **DO NOT touch** `apps/command-center/` theme ‚Äî CC has its own tokens
2. **tokens.css is for portal apps only**: portal, dashboard, support, studio, admin
3. **HSL format**: `228 54% 20%` ‚Äî NO `hsl()` wrapper, NO commas ‚Äî shadcn convention
4. **Sizing in px**: `--button-height-sm: 32px` not just `32`
5. **Font-weight stays numeric**: 400, 500, 600 ‚Äî no px suffix
6. **Skip**: raw colors collection, hacks*, obra-shadcn-docs/*
7. **Preserve order**: keep sections grouped logically, shadcn vars first
8. **Verify after write**: count vars in output, compare with previous count, report delta
